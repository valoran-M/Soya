# Soya langage

Soya is an imperative object oriented langage.

We have two type of stuctures :
- Primitive type
- Object

## Primitive type

### int

```
int x = 42;
```

### char

```
char c = 'c';
```

### bool

b := true | false

```
bool b = true;
```
